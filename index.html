Gesture Command Instructions
The following commands utilize the LEAP Motion Controller

Pause: fist motion over sensor


Play: both hands flat over sensor


Skip to Next Song: fist motion over sensor


Skip to Previous Song: fist motion over sensor


Lower Volume counterclockwise circular motion with index finger extended


Raise Volume: clockwise circular motion with index finger extended


Add/Remove Song from Liked Songs: vertical key tap motion with index finger extended

Voice Command Instructions
The following commands utilize speech detection:

"Queue + {emotion}" : Asks Spotify to add 5 songs to user's queue based on emotion
Emotions Supported : Happy, Sad, Angry, Surprise

"Open/Close Gesture Command" : Opens/Closes gesture command instructions

"Open/Close Voice Command" : Opens/Closes voice command instructions
